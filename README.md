A browser-only application for previewing and printing HTML emails.

## Privacy

- Emails stay in browser memory and are never uploaded or saved.
- HTML emails only. Remote content stays blocked unless you allow supported HTTPS resources for this session.

## Development

```sh
pnpm install
pnpm dev
```
